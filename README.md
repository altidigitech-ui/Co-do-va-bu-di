## CE QUE C'EST

Ce dossier contient la stratégie complète du pivot distribution-first de FoundryTwo. Il remplace l'approche build-first qui a produit 8 signups et 0€ MRR en 3 semaines.

Le loop central :
```
COMMUNAUTÉ → DOULEUR → VALIDATION → BUILD → DISTRIBUTION (VOLUME × CONSTANCE) → REPEAT
```
